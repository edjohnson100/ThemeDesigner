📖 How to Use Theme Designer

💡 Tip: Click any element in the Live Preview to automatically highlight its corresponding color variables!

✨ Creating a New Theme

  1. Select a starting point from the Active Theme dropdown.
  2. Click the ➕ button to duplicate it and provide a new ID.
  3. Edit the colors using the pickers. Changes are saved instantly to memory!

Updating an Existing Theme

  1. Select the theme you want to update from the dropdown.
  2. Adjust the color pickers on the left. Changes are applied instantly.

Saving & Exporting

  • 📤 Export style.css: Generates a full, freshly compiled style.css file combining the base layouts and all your active themes. Save this directly over your existing file.
  • 💾 Save .json: Exports only the currently active theme so you can share it or import it into another project.
